# ISS Grade Encoding Web

This is a GitHub Pages-ready static companion site for ISS grade encoding.

## Included modules

- Dashboard
- My Loads
- Grade Encoding
- My Loads CSV download
- Current load CSV download
- Grade Correction request and tracking

## View modes

The site automatically switches layout based on screen and input style.

- Desktop mode: wide screen with mouse or trackpad, ISS Local App style sidebar, panels, tables, and batch save.
- Tablet and mobile mode: drawer navigation, learner cards, and per-student save so saved learners are not lost when the connection drops.

## Security note

This site must use only the Supabase URL and anon key in `config.js`. Do not add the service role key to this project.

## Deploying to GitHub Pages

Upload all files in this folder to the repository root. In GitHub, open Settings, Pages, then deploy from the main branch root.
